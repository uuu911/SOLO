# 俯视视角赛车竞速游戏 - 产品需求文档

## 1. 产品概述

本项目是一个基于Canvas的俯视视角赛车竞速游戏，提供沉浸式赛车体验。
- 核心功能：环形赛道竞速、漂移系统、AI对手、氮气加速、计时系统
- 目标用户：休闲游戏玩家、赛车游戏爱好者
- 产品价值：提供高品质网页端轻量级赛车游戏体验，无需安装即可游玩

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 玩家 | 无需注册 | 完整游戏体验、本地成绩存储 |

### 2.2 功能模块

1. **游戏主界面**：Canvas渲染区域、HUD信息显示
2. **赛车控制系统**：玩家输入处理、物理引擎、漂移系统
3. **AI系统**：AI赛车路径跟随、避障、速度波动
4. **赛道系统**：多地形赛道、碰撞检测
5. **计时系统**：单圈计时、成绩存储、比赛结束面板

### 2.3 功能详情

| 页面/模块名称 | 模块名称 | 功能描述 |
|----------------|----------|----------|
| 游戏主界面 | Canvas渲染器 | 实时渲染赛道、赛车、特效、HUD界面 |
| 赛车控制 | 物理引擎 | 赛车动力学模拟、地形影响、碰撞检测 |
| 漂移系统 | 漂移机制 | Shift键触发、轮胎印/烟尘特效、速度增益 |
| AI控制 | AI系统 | 路径跟随、避障、速度波动、动态难度 |
| 计时系统 | 计时面板 | 3圈比赛、单圈计时、成绩存储localStorage |
| HUD界面 | 信息显示 | 速度、圈数、时间、氮气条、小地图 |

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深灰色背景，红色/蓝色赛车
- **赛道配色**：沥青灰（路面）、绿色（草地）、土黄色（沙地）
- **字体**：现代无衬线字体，清晰可读
- **布局**：全屏Canvas，HUD位于角落不遮挡赛道
- **动画**：平滑过渡、漂移特效、倒计时动画

### 4.2 界面设计概览

| 模块 | UI元素 |
|------|---------|
| 游戏画面 | Canvas全屏渲染，俯视视角 |
| HUD | 左上角速度、圈数、时间；右上角氮气条；右下角小地图 |
| 暂停界面 | 半透明黑色遮罩，中央"PAUSED"文字 |
| 成绩面板 | 半透明面板，每圈时间、总时间、历史最佳 |

### 4.3 响应式
- 桌面端优先，全屏Canvas渲染
- 键盘控制优化

## 5. 性能要求
- Chrome最新版60Hz刷新率下帧率不低于55FPS
- 可靠的碰撞检测
