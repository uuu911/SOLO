## 1. 产品概述

本项目是一个基于Canvas的平台跳跃游戏原型，所有代码集成在单个HTML文件中，浏览器打开即可游玩。游戏包含玩家控制、平台跳跃、敌人巡逻、金币收集等核心玩法。

- 核心目的：提供一个可直接运行的2D平台跳跃游戏原型，展示完整的游戏开发技术栈
- 目标用户：游戏开发者、学习者、希望快速体验Canvas游戏开发的人员
- 产品价值：作为学习和原型开发的基础模板，包含完整的物理系统、碰撞检测、音效反馈等

## 2. 核心功能

### 2.1 功能模块

1. **游戏主界面**：Canvas渲染游戏画面，包含玩家、平台、敌人、金币等元素

### 2.2 功能详情

| 功能模块 | 功能描述 |
|---------|---------|
| 玩家控制 | 方向键左右移动，空格键跳跃，移动速度240px/s |
| 物理系统 | 重力加速度980px/s²，跳跃初速度-420px/s，流畅不穿模 |
| 平台系统 | 地面平台+5个浮空平台，深灰色，宽度80~160px随机 |
| 敌人系统 | 3个红色巡逻敌人，30x30像素，速度100px/s，碰到重置玩家 |
| 金币系统 | 10个黄色金币，半径8px，收集加10分，收集完胜利 |
| HUD显示 | 左上角显示分数和收集进度，高DPI清晰 |
| 视觉反馈 | 跳跃拉伸、落地压扁、金币闪烁、敌人晃动 |
| 音效反馈 | Web Audio API生成跳跃和收集音效 |
| 响应式布局 | 画布居中，窗口缩放等比例居中，深色背景 |

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：浅蓝色背景(#87CEEB)，棕色地面(#8B4513)，深灰色平台(#333333)，绿色玩家(#00FF00)，红色敌人(#FF0000)，黄色金币(#FFD700)
- **字体**：等宽字体，清晰易读
- **布局**：Canvas居中，深色背景填充剩余空间

### 4.2 界面元素

| 元素 | 位置 | 样式 |
|-----|------|------|
| 游戏画布 | 居中 | 800x600像素 |
| HUD | 左上角 | 白色文字，黑色描边 |
| 胜利文字 | 居中 | 大字号，醒目 |

### 4.3 响应式

- 桌面端：画布等比例缩放，保持居中
- 无滚动条，深色填充
