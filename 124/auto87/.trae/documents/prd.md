# UrbanHustle - 都市生存模拟游戏 产品需求文档

## 1. 产品概述

UrbanHustle是一款现代都市白领生存模拟游戏，玩家需要在30天内平衡金钱、精力、心情、人脉四项核心属性，在城市生活的压力下生存并达成月度目标。

- **核心目标**：通过每天选择不同行动来平衡四项属性，在30天内达成随机目标，体验都市生活的挑战与机遇。
- **目标用户**：喜欢策略模拟游戏、喜欢都市生活题材的玩家。

## 2. 核心功能

### 2.1 核心属性系统
| 属性 | 初始值 | 范围 | 说明 |
|------|--------|------|------|
| 金钱 | 50 | 0-100 | 用于维持生活和社交 |
| 精力 | 50 | 0-100 | 执行工作行动的基础 |
| 心情 | 50 | 0-100 | 影响整体状态 |
| 人脉 | 50 | 0-100 | 社交网络资源 |

### 2.2 功能模块
1. **游戏主界面**：属性面板、行动选择区、日志面板、目标提示
2. **行动系统**：每天三时段（早晨、下午、夜晚），6+行动选择
3. **随机事件系统**：每天30%概率触发随机事件
4. **目标系统**：开局随机抽取月度目标
5. **结局系统**：胜利/失败/破产结局和S/A/B/C/D评价

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|-----------|-------------|---------------------|
| 游戏主页面 | 属性面板 | 实时显示四项属性值及进度条、数字滚动动画 |
| 游戏主页面 | 行动选择区 | 三时段行动按钮，悬停效果 |
| 游戏主页面 | 日志面板 | 记录最近10天的行动和事件 |
| 游戏主页面 | 目标提示 | 显示本月目标 |
| 游戏主页面 | 结果弹窗 | 游戏结束时显示结果和评价 |

## 3. 核心流程

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：深蓝灰色系 (#1a1a2e 到 #16213e)，体现现代都市感
- **辅助色**：金钱(金色#f5ba00)、精力(绿色#4ade80)、心情(粉色#f472b6)、人脉(蓝色#60a5fa)
- **按钮风格**：圆角矩形，渐变背景，悬停时有上浮和阴影效果
- **字体**：现代无衬线字体，清晰易读
- **布局风格**：卡片式布局，清晰的信息层级
- **动画效果**：属性变化数字滚动、按钮悬停效果、弹窗淡入淡出

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI元素 |
|-----------|-------------|-------------|
| 游戏主页面 | 属性面板 | 彩色进度条、数值显示、数字滚动动画 |
| 游戏主页面 | 行动选择区 | 网格布局按钮、悬停上浮效果、点击反馈 |
| 游戏主页面 | 日志面板 | 滚动区域、时间线样式、渐变背景 |
| 游戏主页面 | 结果弹窗 | 居中弹窗、渐变背景、评价等级显示 |

### 4.3 响应式
桌面优先设计，适配1280×720分辨率，核心元素无需滚动即可完整显示。

### 4.4 视觉细节
- 使用渐变背景营造都市夜景氛围
- 卡片阴影增加层次感
- 平滑过渡动画提升交互体验
- 属性变化时的微动画反馈
